

##  一键部署 (VPS)

### 方式 1：自动安装脚本（推荐）

直接复制以下命令到服务器终端运行即可：

```bash
bash <(wget -qO- https://raw.githubusercontent.com/tony-wang1990/King-Detective/main/scripts/install.sh)
```

脚本会自动完成：

- 环境检测 (Docker/Compose)
- 拉取最新镜像
- 启动服务


## 💻 访问应用
